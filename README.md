# practice-django-react
1.Начало: Развернутый сервер с подключенным unfold (админка работает) и django rest правдо покачто пустые сериализаторы но это исправится
2.Создание моделей: Создание моделей в Shop и подключение DjangoRest
3.Создание фронта: Дополнительное изучение материала прокачка REST на стороне сервера, добавление react и axios, а также пробное связывание и вывод на фронт части информации с сервера ура
4.Страница товаров: прокачка каталога (показываются картинки) а также переход на отделбные карточки товаров.
5.Вход и регистрация: дополнительное приложение в джанго где хранится наша модель, и связь с фронтом в качестве пост запросов с форм.
